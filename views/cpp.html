Introduction to C++

C++ is a general-purpose, high-performance programming language.

What is C++?

C++ is an extension of the C programming language with additional features, making it suitable for a wide range of applications.

Namespace in C++

Namespaces prevent name collisions and help organize code into logical groups.


#include 

namespace mynamespace {
    void display() {
        std::cout << "This is mynamespace!" << std::endl;
    }
}

int main() {
    mynamespace::display();
    return 0;
}

Object-Oriented Programming (OOP) Concepts in C++

C++ is widely used for object-oriented programming, which focuses on organizing code into objects and classes.

Encapsulation

Encapsulation refers to bundling data and methods within a class.


#include 

class Account {
private:
    double balance;

public:
    void setBalance(double b) { balance = b; }
    double getBalance() { return balance; }
};

int main() {
    Account acc;
    acc.setBalance(1000.0);
    std::cout << "Balance: " << acc.getBalance() << std::endl;
    return 0;
}

Polymorphism

Polymorphism allows objects to be treated as instances of their parent class.


#include 

class Print {
public:
    void show(int i) {
        std::cout << "Integer: " << i << std::endl;
    }
    void show(double d) {
        std::cout << "Double: " << d << std::endl;
    }
};

int main() {
    Print p;
    p.show(5);       // Calls show(int)
    p.show(3.14);    // Calls show(double)
    return 0;
}

Abstract Classes

Abstract classes cannot be instantiated and usually contain at least one pure virtual function.


#include 

class Shape {
public:
    virtual void draw() = 0;  // Pure virtual function
};

class Circle : public Shape {
public:
    void draw() {
        std::cout << "Drawing Circle" << std::endl;
    }
};

int main() {
    Circle c;
    c.draw();
    return 0;
}

Access Modifiers in C++

C++ provides three access modifiers: private, public, and protected.


#include 

class Box {
private:
    double length;

public:
    void setLength(double len) { length = len; }
    double getLength() { return length; }
};

int main() {
    Box b;
    b.setLength(5.0);
    std::cout << "Length of box: " << b.getLength() << std::endl;
    return 0;
}
